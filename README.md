# VORP NUI Event
> Welcome to our NUI Tutorial Event!

## How to install
* Download this repo
* Copy and paste `vorp_status` folder to `resources/vorp_status`
* Add `ensure vorp_status` to your `server.cfg` file
* Now you are ready to get coding!
